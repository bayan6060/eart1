1. Install Netbeans IDE to Windows 7/10 machine
2. load the project in the IDE
3. Set the path from where to load the text file for encryption
4. Enable the corresponding encrytion (EART, AES, DES) 
5. Run the code.
6. For decrytion, enable the decryption algorithm by uncommenting.
7. Define the path of decryption and run the code.
